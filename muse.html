iOS Application

Portfolio

Muse is a web-app that makes every commute a creative experience while also bringing public, user-generated art to DC's metro stations.

Each week users are given a theme based on a quote from a DC native. Using this theme as a guide, each user can create their own works of art while traveling through the metro. Submitted works can be favorited by users on the gallery page. The more 'likes' a submitted work gets, the larger the image appears on 'digital canvases', set up in metro stations across the city.
